# 🎮 Flappy Bird Clone – Fantasy Theme (Unity 2D)

A Flappy Bird-style game developed in Unity, featuring a custom fantasy-inspired theme and original pixel-style assets.

This project focuses on recreating core game mechanics while experimenting with visual design and game feel.

---

## 🚀 Gameplay Overview

- 🎮 Classic tap-to-fly mechanics  
- 🧩 Obstacle avoidance gameplay  
- 📈 Score system based on progression  
- 💀 Instant restart on Game Over  

---

## 🎨 Visual Concept

Instead of the traditional bird and pipes:

- 🚗 The player controls a **flying car**  
- 🐍 Obstacles are **serpent-like enemies**  
- 🏰 Background inspired by a **fantasy castle environment**  

This approach allowed me to combine **game development + pixel art design** into a single project.

---

## 🛠️ Tech Stack

- Unity (2D)  
- C#  
- Game physics & collision system  

---

## 🕹️ Controls

- Left Click → Make the player jump/fly
